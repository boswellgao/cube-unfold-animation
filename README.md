# 🎲 正方体展开动画

> 面向小学生的 Web 3D 教学动画，展示正方体从立体形态逐步展开为平面展开图的过程。

---

## ✨ 功能特色

| 功能 | 说明 |
|------|------|
| 🟥🟦🟩🟨🟧🟪 六色面 | 每个面颜色各异，直观易辨 |
| 🏷️ 中文面名标注 | 前、后、左、右、上、下 |
| 🔄 展开 / 还原动画 | 铰链式旋转，物理正确，分两阶段展开 |
| 🖱️ 鼠标交互 | 拖拽旋转视角，滚轮缩放 |
| ⚡ 速度调节 | 慢速便于观察，快速便于演示 |
| 📝 步骤提示 | 动画过程中显示中文说明 |

## 🚀 使用方法

1. 直接在浏览器中打开 `index.html`，**无需安装任何工具或服务器**。
2. 点击 **📐 展开** 按钮，观看正方体六个面逐步翻转为十字形平面展开图。
3. 点击 **🔄 还原** 按钮，将展开图重新组合回正方体。
4. 拖动速度滑块调整动画快慢。
5. 用鼠标拖拽旋转视角，从不同角度观察展开过程。

## 🧩 展开方式说明

采用经典**十字形展开**：

```
   [上]   ← z=2
   [前]   ← z=1
[左][下][右] ← z=0, 底面居中
   [后]   ← z=-1
```

动画分两阶段：
1. **第一阶段**：前、后、左、右四个侧面同时向外翻转 90°，与底面铺平。
2. **第二阶段**：顶面沿前面的上边缘再翻转 90°，完成展开。

## 🛠️ 技术栈

- **HTML5 + CSS3 + Vanilla JavaScript**
- **Three.js r134**（CDN 引入，无需安装）
- **OrbitControls**（Three.js 内置插件）
- 无构建工具，浏览器直接打开即可运行
- 兼容 Chrome、Firefox、Edge、Safari 等现代浏览器
